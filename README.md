# APCSA Game Project: Stuyvesant vs. the Evil Lunch Ladies

Developers: Marc Haddad, Leon Wong

To play our game,
1) Git clone the respository into your local machine by typing this command into your terminal:
                          git clone git@github.com:orontis/waltwhitman.git
2) Compile the code by using the javac command followed by the name of each file.
3) Run the game by using the command:     java Game
4) Enjoy
(To exit the came, use Ctr + C or Command + C)
